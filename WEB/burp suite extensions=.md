## 🧩 Burp Suite Extensions

***

Burp Suite's functionality can be significantly extended using community and custom extensions from the [BApp Store](https://portswigger.net/bappstore). Below are some of the most popular and useful extensions for web application penetration testing.

  

#### 🔍 Scanning & Automation

***

[ActiveScan++](https://portswigger.net/bappstore/3123d5b5f25c4128894d97ea1acc4976)

Enhances Burp's built-in active scanner with additional checks and payloads.

  

[Backslash Powered Scanner](https://portswigger.net/bappstore/9cff924534f943579277d3370591f422)

Performs advanced scanning and payload delivery using injection techniques.

  

[Param Miner](https://portswigger.net/bappstore/1759492a985740638b97d57ae4fbf203)

Automatically discovers hidden, unlinked, and undocumented parameters in web apps.

  

#### 🔐 Authorization & Session Testing

***

[Autorize](https://portswigger.net/bappstore/f9bbac8c4acf4aefa4d7dc92a991af2f)

Detects broken access controls and authorization flaws with automated session switching.

  

[AuthMatrix](https://portswigger.net/bappstore/matrix)

Streamlines role-based access control testing and session management.

  

[AutoRepeater](https://portswigger.net/bappstore/f89f2837c22c4ab4b772f31522647ed8)

Repeats requests with customizable match/replace rules; great for auth testing.

  

#### 📦 Application-Specific

***

[J2EEScan](https://portswigger.net/bappstore/2299471f55c8457788647087019808a6)

Adds tests for Java EE-based vulnerabilities.

  

[Software Vulnerability Scanner](https://portswigger.net/bappstore/c26b152f2393433695b4528170366b5b)

Detects CVEs in platforms like WordPress, Joomla, Drupal, etc.

  

#### 🛡️ Crypto & Token Handling

***

[JWT Editor](https://portswigger.net/bappstore/f5c3511110824b4f8d93a43160a34032)

Full-featured editor and validator for JSON Web Tokens (JWTs).

  

#### 📁 Upload & File-Based Vulnerabilities

***

[Upload Scanner](https://portswigger.net/bappstore/b2244cbb6903422fa97430fd38208468)

Scans upload forms for issues like weak extension checks, path traversal, etc.

  

#### 🕵️ Recon & Discovery

***

[JS Link Finder](https://portswigger.net/bappstore/f032258d60234407817541b525d19e07)

Parses JavaScript files for URLs/endpoints—useful for hidden paths and XSS.

  

[Retire.js](https://portswigger.net/bappstore/0a3c262a054549fca32752140a3bd095)

Detects vulnerable JavaScript libraries.

🔗 [Official site](https://retirejs.github.io/retire.js/)

  

[WSDLer](https://portswigger.net/bappstore/5940b5c5453341b7a542476065522f95)

Enumerates and scans SOAP web services via WSDL parsing.

  

#### 🧪 HTTP/Protocol Abuse

***

[HTTP Request Smuggler](https://portswigger.net/bappstore/9a46cce95185458c92f15f3e29f3f721)

Detects request smuggling vulnerabilities like CL.TE and TE.CL.

  

#### 🧰 Tools & Utility Extensions

***

[Logger++](https://portswigger.net/bappstore/470b7057b869415bc2f347354a8397a6)

Enhanced logging for requests, responses, IPs, headers, and more.

🔗 [Docs](https://github.com/nccgroup/LoggerPlusPlus) | [Filters](https://github.com/nccgroup/LoggerPlusPlus/wiki/Filters)

  

[Hackbar](https://portswigger.net/bappstore/6513e9d892304383844d187ed23b5d30)

Manually test inputs, craft payloads, and edit requests from a user-friendly UI.

  

[JSON Pretty Print](https://github.com/doyensec/burp-extensions) (no direct link available)  

Formats and beautifies JSON data in Burp views.

  

[Broken Link Hijacking](https://portswigger.net/bappstore/a3130541257449a5b766f6894562089e)

Detects hijackable broken links and open redirect issues.

  

#### 🔒 SSL/TLS Testing

***

[SSL Scanner](https://portswigger.net/bappstore/474b3c57c53641b8a257c21b58941f62)

Audits HTTPS endpoints for SSL/TLS misconfigurations and weak ciphers.
