# Penetration testers, also known as ethical hackers, need a combination of technical and soft skills. 

Technical skills

- **Coding**
    
    Penetration testers use coding skills in multiple languages to penetrate networks and systems. 
    
- **Networking**
    
    Penetration testers need to understand network components, such as firewalls, virtual local area networks (VLANs), and network switches. 
    
- **Security**
    
    Penetration testers need to understand how to protect computer systems and networks, and how to implement security measures. 
    
- **Cybersecurity**
    
    Penetration testers need to understand cybersecurity and how to identify and fix vulnerabilities in a system. 
    
- **Computer forensics**
    
    Penetration testers need to understand how to analyze systems and recover information. 
    

Soft skills

- **Communication**: Penetration testers need to be able to communicate their findings to system owners and administrators. 
- **Teamwork**: Penetration testers need to be able to work well with others. 
- **Report writing**: Penetration testers need to be able to write reports that clearly communicate their findings. 

Education and certifications

- Penetration testers typically have a degree in computer science, IT, or cybersecurity. 
- They may also earn professional certifications to demonstrate their skills
# Security engineers need a variety of skills to protect an organization's systems, networks, and data from security threats. These skills include: 

- **Incident response**: The ability to identify, manage, and mitigate security breaches and cyber attacks 
- **Penetration testing**: The ability to simulate cyberattacks to identify security weaknesses 
- **Vulnerability assessment**: The ability to identify and mitigate security risks using vulnerability assessment tools 
- **Network security**: The ability to design, implement, and maintain security systems to protect an organization's IT infrastructure 

- **Problem-solving**: The ability to think on your feet and find and implement solutions to security threats 

- **Risk assessment**: The ability to assess and manage the risk of security threats 

- **Cryptography**: The ability to understand encryption techniques and other cryptographic methods 

- **Firewall configuration**: The ability to understand firewall configurations and intrusion detection systems 

- **Threat modeling**: The ability to model threats and assess the risk they pose 

- **Communication**: The ability to communicate effectively with other team members 

Other skills that security engineers may need include: 

- Computer networking

- Operating systems

- IT support

- Computer hardware and software

- Computer programming

- Virus protection software

- Data management

- Security standards

- Project management
