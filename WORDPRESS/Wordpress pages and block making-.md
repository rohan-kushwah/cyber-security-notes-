# 🌐 WordPress Sample Pages with Media and Explanation

  

This guide provides 3 sample pages for WordPress with explanations, YouTube links, and images. These examples can help you understand how to build real-world pages in WordPress using the Gutenberg editor or Elementor.

  

---

  

## 📄 Page 1: About Us

  

### 🎯 Purpose:

To introduce the company/individual and give an overview of the brand or mission.

  

### ✍️ Sample Content:

  

**Title:** About Us

  

**Content:**

```

Welcome to XYZ Solutions!

  

We are dedicated to providing innovative web solutions that help businesses grow online. Our team specializes in WordPress development, SEO, and digital marketing.

```

  

**YouTube Embed (Company Overview):**

```

[![Watch Company Overview](https://img.youtube.com/vi/tgbNymZ7vqY/0.jpg)](https://www.youtube.com/watch?v=tgbNymZ7vqY)

```

  

**Image:**

```

![Team Photo](https://via.placeholder.com/600x300?text=Our+Team)

```

  

**How to Add in WordPress:**

1. Go to **Pages > Add New**.

2. Title the page **About Us**.

3. Use **Paragraph**, **Image**, and **Video** blocks.

4. Paste the YouTube link and WordPress will automatically embed it.

5. Click **Publish**.

## 📄 Page 3: Contact Us

  

### 🎯 Purpose:

To allow visitors to get in touch via form, phone, or location map.

  

### ✍️ Sample Content:

  

**Title:** Contact Us

  

**Content:**

```

We would love to hear from you!

  

📧 Email: contact@xyzsolutions.com  

📞 Phone: +91-1234567890  

📍 Location: New Delhi, India

```

  

**Image:**

```

![Office](https://via.placeholder.com/600x300?text=Our+Office)

```

  

**Embed Google Map:**

Use the Custom HTML block to add Google Maps embed code:

```html

<iframe src="https://www.google.com/maps/embed?pb=!1m18..." width="100%" height="300" style="border:0;" allowfullscreen="" loading="lazy"></iframe>

```

  

**How to Add in WordPress:**

1. Title the page **Contact Us**.

2. Use **Paragraph**, **Image**, and **Custom HTML** blocks.

3. Paste your Google Maps embed code.

4. Use a contact form plugin like WPForms or Contact Form 7.

5. Click **Publish**.

  

---

  

## ✅ Tips for All Pages

  

- Use **Heading blocks** for structure (H2, H3, etc.)

- Use **Media blocks** to embed YouTube or add images

- Ensure pages are added to **Menus** via Appearance > Menus

- Install **Elementor** for more drag-and-drop control
